ML-Based Cochlear Implant Path Planning

Expanded a MATLAB-based cochlear implant simulation algorithm by incorporating Python and machine learning. Integrated 3D-printed temporal bone models and sensor-instrumented guide wires to collect real-world insertion data. This hybrid model supports data-driven optimization for surgical path accuracy and patient-specific risk reduction.

Tools: MATLAB, Python, SolidWorks, ITK-SNAP, 3D Printing

3D Anatomical Ear Model for Surgical Education

Segmented MRI scans using ITK-SNAP and built a detailed anatomical ear model including the cochlea, skull base, and auditory canal. Assembled and printed using SolidWorks for patient and clinical education. The model was deployed as part of a surgical planning and outreach initiative.
